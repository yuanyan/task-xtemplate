# task-xtemplate
> KISSY XTemplate Compiler.

## The "xtemplate" task

### Usage Examples

```js
var xtemplate = new (require('task-xtemplate'))
xtemplate.run(inputs, options, logger)
```

### Options

#### options.version
Type: `string`

Your KISSY version, default is latest version.
